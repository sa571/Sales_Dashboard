# Executive Summary — Sales Performance Dashboard
## This dashboard provides a high-level overview of sales performance, customer behavior, product contribution, and geographic distribution across two years (2018–2019). It transforms over 500,000 transactional records into actionable business insights that support strategic decision-making.
